# 🚀 AI Job Market Analysis Dashboard

## 📊 Project Overview

This project analyzes the AI job market using Power BI. It provides insights into job demand, salary trends, experience levels, and work types.

---

## 🔥 Key Features

* Job Role Demand Analysis
* Salary Insights by Role
* Work Type Distribution (Remote/Hybrid/Onsite)
* Experience Level Analysis
* Hiring Urgency Trends
* Country-wise Job Distribution

---

## 🛠 Tools Used

* Power BI
* Python (Pandas, NumPy)
* Excel

## 📂 Dataset

Dataset includes:

* Job Titles
* Salaries
* Experience Levels
* Work Type
* Country

---

## 💡 Insights

* AI Engineers and ML Engineers have highest salaries (~140K)
* Demand is highest for Business Analysts and AI Engineers
* Remote jobs slightly dominate the market
